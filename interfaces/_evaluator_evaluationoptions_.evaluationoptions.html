allowUnknownFilters

allowUnknownFilters: boolean
  • Defined in src/evaluator/EvaluationOptions.ts:26

Whether to allow unknown filters. If set to false, then an UnknownFilterError will be thrown if an unknown filter is found. Unknown means, that there is no function in the filters object for the specified name.

allowUnknownProperties

allowUnknownProperties: boolean
  • Defined in src/evaluator/EvaluationOptions.ts:12

Whether to allow unknown properties. If set to false, then an UnknownPropertyError will be thrown if an unknown property is found. Unknown means, that a pointed property is not present on the substitution object.
